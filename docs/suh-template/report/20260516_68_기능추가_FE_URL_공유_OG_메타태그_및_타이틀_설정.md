# URL 공유 시 OG 메타태그 및 타이틀 설정

## 개요

카카오톡, 디스코드, 슬랙, 트위터(X) 등 SNS에서 URL 공유 시 게임 정보가 정상적으로 미리보기 되도록 `index.html`의 타이틀과 OG/Twitter Card 메타태그를 추가했다. 썸네일 이미지(`og-image.png`)도 함께 등록하여 시각적 프리뷰를 제공한다.

## 변경 사항

### 메타태그 추가 (index.html)
- `<title>`: `pickerpicker-init` → `PickerPicker — 한글 음절로 리듬을 타는 생존 리듬게임`으로 변경
- `og:type`, `og:url`, `og:title`, `og:description`, `og:image` 추가
- `twitter:card`, `twitter:title`, `twitter:description`, `twitter:image` 추가

### 썸네일 이미지 등록 (public/og-image.png)
- 게임 소개 이미지를 `public/` 에 배치
- OG/Twitter Card 이미지 URL: `http://suh-project.synology.me:3010/og-image.png`

## 주요 구현 내용

- Twitter Card 타입은 `summary_large_image`로 설정하여 넓은 이미지 카드 형태로 노출
- OG URL은 실서버 주소(`http://suh-project.synology.me:3010`)로 절대경로 지정
- 이미지 역시 절대경로로 지정하여 크롤러가 상대경로 해석 실패하는 문제 방지

## 주의사항

- 배포 도메인 변경 시 `og:url`, `og:image`, `twitter:image`의 도메인도 함께 수정 필요
- SNS 플랫폼의 캐시로 인해 메타태그 반영까지 최대 수십 분 소요될 수 있음
- 카카오톡은 [카카오 공유 디버거](https://developers.kakao.com/tool/clear/og)로 캐시 초기화 가능
