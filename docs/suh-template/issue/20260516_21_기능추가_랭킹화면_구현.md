📝 현재 문제점
---

- `RankingScreen` 컴포넌트가 "미구현" 텍스트만 표시하는 빈 껍데기 상태
- 시작화면에서 랭킹 버튼을 눌러도 실제 랭킹 데이터를 볼 수 없음
- `App.tsx`에서 `RankingScreen`으로 전환 시 돌아가기(`onBack`) prop 연결이 누락됨

🛠️ 해결 방안 / 제안 기능
---

- `GET /ranking?limit=20` API 연동하여 실제 랭킹 데이터 표시
- 순위 / 닉네임 / 점수 / 스테이지 / 콤보 5컬럼 테이블 구현 (DaisyUI table)
- 로딩 상태 처리 (DaisyUI loading spinner)
- 빈 데이터 상태 처리 ("아직 랭킹 데이터가 없습니다")
- 돌아가기 버튼 → 시작화면으로 전환

⚙️ 작업 내용
---

- `RankingScreen.tsx` — 실제 랭킹 테이블 UI 및 API 연동 구현
- `App.tsx` — `RankingScreen`에 `onBack` prop 추가 및 연결
