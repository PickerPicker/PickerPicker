# 🚀[기능개선][홈화면] 홈화면 버전 정보 표시 및 UI 개선

- **라벨**: 작업전
- **담당자**: Cassiiopeia

📝 현재 문제점
---

- 홈화면에 버전 정보가 표시되지 않아 빌드 버전 확인이 불가능함
- 로그아웃 버튼의 높이/글씨 크기/폰트가 다른 버튼과 일치하지 않아 UI 일관성 부족
- 닉네임 표시 아래 불필요한 패딩이 과도하게 적용되어 있음
- 카드 영역 전체 패딩이 커서 화면 공간 낭비

🛠️ 해결 방안 / 제안 기능
---

- `vite.config.ts`에서 빌드 시 `version.yml`을 파싱하여 `__APP_VERSION__` 전역 변수로 주입
- 홈화면 타이틀 하단에 `2026 InCerthon v{버전}` 형태로 한 줄 표시 (`src/components/StartScreen.tsx`)
- 로그아웃 버튼을 `btn-lg text-lg monospace` 폰트로 통일하여 다른 버튼과 동일한 높이/크기 적용
- 닉네임 표시 영역 하단 마진 축소 (`mt-1` → `marginTop: 2px`)
- 카드 패딩 축소 (`px-16 py-12` → `px-10 py-8`)

⚙️ 작업 내용
---

- `vite.config.ts` — yaml 파싱으로 버전 주입 (`define.__APP_VERSION__`)
- `src/vite-env.d.ts` — `__APP_VERSION__` 전역 타입 선언 추가
- `src/components/StartScreen.tsx` — 버전 표시, 로그아웃 버튼 스타일, 패딩 조정
