# 🚀[기능추가][연습모드] 난이도별 3스테이지 연습 환경 추가

- **라벨**: 작업전
- **담당자**: Cassiiopeia

📝 현재 문제점
---

- 현재 본 게임은 15스테이지 연속 진행 + 게이지 0이면 즉시 게임 오버되는 생존형 구조
- 사용자가 특정 난이도 구간만 골라서 부담 없이 연습할 수단이 없음
- 8키 진입(7~9), 복수 단어(10~12), 고난도(13~15) 같은 후반 난이도를 처음부터 도달해야만 익힐 수 있음
- 게임플레이 경험이 있는 사용자에게 난이도 단계별 반복 연습 환경 필요

🛠️ 해결 방안 / 제안 기능
---

- 홈화면(StartScreen)에 **[연습모드]** 메인 버튼 추가
  - **본 게임 1회 이상 플레이한 사용자에게만 노출** (`localStorage.pickerpicker_best` 키 존재 여부로 판별)
  - 무경험 사용자는 본 게임 흐름을 먼저 익히도록 유도
- 연습모드 화면: 난이도 5단계 카드 노출
  - 1단계 입문 / 2단계 확장 / 3단계 8키 진입 / 4단계 복수 단어 / 5단계 고난도
  - 각 카드 클릭 시 해당 난이도 3개 스테이지 연속 플레이
- 연습 플레이 규칙
  - **게이지 시스템 미사용** — MISS 나도 게임 오버 없음, 끝까지 플레이 보장
  - **결과 저장 일절 없음** — `localStorage` best 갱신 ✗, 서버 `POST /players/result` 호출 ✗, 랭킹/playCount 영향 ✗
  - 화면 상 점수/판정은 정상 표시 (피드백 용도)
- 3스테이지 완료 후
  - 간단한 "○○ 연습 완료" 화면 표시
  - [다른 난이도 선택] → 연습모드 메뉴 / [메인으로] → StartScreen
- 추후 JSON 데이터셋 리팩토링(스테이지 랜덤화) 작업이 들어가도 난이도별 3스테이지 그룹 구조는 변함없음 → 본 설계도 그대로 유효
