# ⚙️[기능추가][FE] URL 공유 시 OG 메타태그 및 타이틀 설정

- 라벨: 작업전
- 담당자: Cassiiopeia

---

📝 현재 문제점
---

- `index.html`의 `<title>`이 `pickerpicker-init`으로 설정되어 있어 브라우저 탭과 링크 공유 시 의미 없는 텍스트가 노출됨
- OG(Open Graph) 메타태그가 전혀 없어 카카오톡, 디스코드, 슬랙, 트위터 등 SNS에서 URL 공유 시 프리뷰가 비어있거나 깨짐
- Twitter Card 메타태그도 부재하여 트위터/X 공유 시 정상적인 카드 미리보기 불가

🛠️ 해결 방안 / 제안 기능
---

- `index.html` `<title>`을 게임명/설명으로 변경
- OG 메타태그 추가: `og:title`, `og:description`, `og:type`, `og:url`, `og:image`
- Twitter Card 메타태그 추가: `twitter:card`, `twitter:title`, `twitter:description`, `twitter:image`
- `og:image` / `twitter:image`에 썸네일 이미지 연결 (이미지 준비 후 경로 지정)

⚙️ 작업 내용
---

- `index.html` 타이틀 수정
- OG 메타태그 블록 추가
- Twitter Card 메타태그 블록 추가
- `public/` 또는 `src/assets/`에 og-image 파일 배치 및 경로 연결
