# ⚙️[기능추가][Game] 스테이지 미리보기 화면 및 본게임 화면 구현

- 라벨: 작업전
- 담당자: LOOPARAM

---

📝 현재 문제점
---

- `GameScreen.tsx`가 stub 상태 ("게임 화면 미구현" 텍스트만 표시)로, 실제 게임을 플레이할 수 없다.
- 리듬 미리보기 화면과 본게임 화면 모두 미구현이다.

🛠️ 해결 방안 / 제안 기능
---

- 스테이지 미리보기 화면: BPM에 맞춰 리듬 큐를 자동으로 강조하고, 키보드 매핑을 동기화하여 표시한다.
- 본게임 화면: 노트가 오른쪽에서 판정선 방향으로 이동하며, 플레이어가 키보드를 입력하면 PERFECT/GOOD/MISS 판정이 표시된다.
- 두 화면은 자동 전환된다 (미리보기 1회 완료 → 본게임 자동 시작).

⚙️ 작업 내용
---

- [ ] 타입 정의 추가 (`StageData`, `GamePhase`, `GameStat`, `JudgmentType`)
- [ ] 스테이지 JSON 데이터 `public/` 폴더에 배치
- [ ] `GameHeader` 컴포넌트 구현 (Stage, 이번 단어, 게이지 바, Score)
- [ ] `KeyboardDisplay` 컴포넌트 구현 (8키 매핑, 미리보기/본게임 공통)
- [ ] `PreviewStage` 컴포넌트 구현 (BPM 박자 자동 강조)
- [ ] `JudgmentDisplay` 컴포넌트 구현 (PERFECT/GOOD/MISS + N combo)
- [ ] `NoteTrack` 컴포넌트 구현 (노트 오른쪽→판정선 이동)
- [ ] `PlayStage` 컴포넌트 구현 (판정 로직: ±70ms PERFECT, ±140ms GOOD)
- [ ] `GameScreen` 컨테이너 구현 (JSON 로딩, 스테이지 상태 관리, 화면 전환)
