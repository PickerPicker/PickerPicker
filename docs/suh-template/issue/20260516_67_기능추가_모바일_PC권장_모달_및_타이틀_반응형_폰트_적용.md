# ⚙️[기능추가][UI] 모바일 접근 시 PC 권장 모달 표시 및 타이틀 반응형 폰트 적용

- 라벨: 작업전
- 담당자: Cassiiopeia

📝 현재 문제점
---

- PickerPicker는 키보드 조작 기반 리듬 게임으로 모바일 환경을 지원하지 않음
- 모바일 브라우저로 접근 시 아무런 안내 없이 게임 화면이 그대로 노출됨
- `StartScreen.tsx`의 h1 타이틀(`text-5xl`)이 모바일 소형 화면(412px 이하)에서 레이아웃을 벗어나 렌더링됨

🛠️ 해결 방안 / 제안 기능
---

- 모바일 접근(viewport width < 768px) 감지 시 PC 플레이 권장 모달 표시
  - 게임 진입을 막지 않고 "확인" 버튼으로 dismiss 가능
  - `sessionStorage` 기반으로 세션 내 1회만 표시
- h1 타이틀 반응형 폰트: `text-3xl`(모바일) / `text-5xl`(sm 이상)

⚙️ 작업 내용
---

- `src/components/common/MobileWarningModal.tsx` 신규 생성
- `src/components/StartScreen.tsx` h1 Tailwind 클래스 반응형 수정
- `src/App.tsx` 모바일 감지 state + MobileWarningModal 렌더 연결
