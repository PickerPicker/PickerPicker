📝 현재 문제점
---

- 채보 게임에서 모든 판정(PERFECT/GOOD/MISS)에 동일한 효과음(`sfx_note.mp3`)이 재생됨
- 정타와 미스를 소리로 구분할 수 없어 게임 피드백이 불명확함

🛠️ 해결 방안 / 제안 기능
---

- 채보1.mp3(박수 소리)를 정타음(`sfx_note_hit.mp3`)과 미스음(`sfx_note_miss.mp3`)으로 분리 등록
- PERFECT/GOOD 판정 시 → `sfx_note_hit.mp3` 재생
- MISS 판정 시 → `sfx_note_miss.mp3` 재생
- (현재는 두 파일 동일, 추후 미스 전용 사운드로 교체 가능하도록 파일 분리만 먼저 적용)

⚙️ 작업 내용
---

- `public/audio/sfx_note_hit.mp3` / `sfx_note_miss.mp3` 파일 추가
- `useAudio.ts`: `playNoteSfx` → `playHitSfx` + `playMissSfx` 분리
- `PlayStage.tsx`: `onNoteSfx` prop → `onHitSfx` + `onMissSfx` 로 교체, 판정 결과에 따라 분기 호출
- `GameScreen.tsx`: props 인터페이스 및 전달 업데이트
- `App.tsx`: `audio.playHitSfx` / `audio.playMissSfx` 연결
