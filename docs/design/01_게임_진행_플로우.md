# 게임 진행 플로우

## 개요

PickerPicker는 15개의 스테이지를 **중단 없이 연속으로 진행**하는 생존형 리듬게임이다.
스테이지별 클리어 화면이나 안내 화면은 없고, 게이지가 0이 되거나 15스테이지를 전부 클리어했을 때만 결과 화면을 표시한다.

---

## 전체 흐름

```
시작 화면
↓
Stage 1 리듬 미리보기
↓
Stage 1 플레이
↓
Stage 2 리듬 미리보기
↓
Stage 2 플레이
↓
...
↓
Stage 15 플레이
↓
ALL CLEAR 결과 화면
```

중간에 게이지가 0이 되면:

```
플레이 중 게이지 0
↓
GAME OVER 결과 화면
```

---

## 화면 상태 정의

| 상태 | 설명 |
| --- | --- |
| `READY` | 시작 화면 |
| `PREVIEW` | 리듬 미리보기 화면 |
| `PLAYING` | 실제 플레이 화면 |
| `RESULT` | 게임 결과 화면 |

> `STAGE_INTRO`, `COUNTDOWN`, `STAGE_CLEAR` 상태는 사용하지 않는다.

---

## 스테이지 전환 방식

- 스테이지 클리어 시 중간 결과 화면 없이 즉시 다음 스테이지 미리보기로 전환
- 전환 시 화면에 `STAGE N` 텍스트를 짧게 표시하는 것만으로 충분
- 카운트다운은 생략 (노트가 날아오는 것 자체가 준비 신호 역할)

---

## 게이지 표시

- 게이지는 숫자 대신 **시각적 바(bar) 형태**로 표시
- 리듬 미리보기 화면과 플레이 화면 모두에서 항상 노출

```
LIFE ████████░░░░  Score 3200
```

- 스테이지 간 게이지는 초기화하지 않고 그대로 유지된다.

---

## 결과 화면

결과 화면은 두 경우에만 표시된다.

### GAME OVER (게이지 0)

```
GAME OVER

도달 스테이지     STAGE 9
최종 점수         12,400
최대 콤보         11
PERFECT          82
GOOD             24
MISS             9
정확도            92%

역대 최고 기록
  최고 점수       18,900
  최고 스테이지   STAGE 12
  최고 콤보       17
  플레이 횟수     8회
```

### ALL CLEAR (Stage 15 완료)

```
ALL CLEAR

최종 점수         31,200
최대 콤보         23
PERFECT          180
GOOD             36
MISS             4
정확도            97%

역대 최고 기록
  최고 점수       31,200
  최고 콤보       23
```

---

## 정확도 계산

```
정확도 = (PERFECT + GOOD) / 전체 입력 수 * 100
PERFECT RATE = PERFECT / 전체 입력 수 * 100
```
