# 본 게임 화면

## 개요

실제 플레이 화면이다. 미리보기와 동일한 레이아웃을 유지하되, 사용자가 직접 키보드를 입력한다.
BPM에 맞춰 노트가 이동하며, 사용자는 현재 강조된 음절에 매핑된 키를 눌러야 한다.

---

## 화면 레이아웃

```
┌────────────────────────────────────────────┐
│  STAGE 7     LIFE ██████░░░░    SCORE 5400  │
│                                   COMBO 6   │
├────────────────────────────────────────────┤
│              이번 단어                      │
│              초콜릿                         │
│                                            │
│  판정선 │        ← [초]  ← [콜]  ← [릿]   │
│                                            │
│          PERFECT!                          │
│                                            │
├────────────────────────────────────────────┤
│               키보드 매핑                   │
│  [A] [S] [D] [F]    [J] [K] [L] [;]        │
│   초  추  차  콜     릿  린  룻  롯          │
└────────────────────────────────────────────┘
```

---

## 영역별 명세

### 1. 상단 상태 바

| 항목 | 설명 |
| --- | --- |
| 스테이지 | `STAGE N` |
| 생존 게이지 | 바(bar) 형태, 항상 표시 |
| 점수 | `SCORE N` |
| 콤보 | `COMBO N` (PERFECT 연속 횟수) |

### 2. 이번 단어

- 미리보기와 동일하게 표시
- 단어 2개 이상이면 줄바꿈 나열

### 3. 플레이 영역 (노트)

MVP와 고도화 버전 두 가지 방식 중 선택:

**MVP — 현재 음절 강조 방식**

```
[커] [커] [커] [커] [피] [피]
  ↑  (현재 처리할 음절 강조)
```

구현이 단순하고 화면이 깔끔하다.

**고도화 — 오른쪽 → 왼쪽 노트 이동 방식**

```
판정선 │        ← [커]     ← [커]     ← [피]
```

노트가 오른쪽에서 날아와 판정선에 도달할 때 입력한다. 리듬게임 느낌이 강하지만 구현 복잡도가 높다.

> 첫 구현은 MVP 방식으로 시작하고, 이후 고도화 방식으로 전환한다.

### 4. 판정 결과 표시

입력 즉시 화면에 판정 결과를 표시한다.

| 상황 | 화면 표시 |
| --- | --- |
| 정답 입력 | `PERFECT!` 또는 `GOOD!`, Combo +1, 점수 증가 |
| 오답 입력 | `MISS!`, Combo 초기화 |

완료된 음절은 체크 또는 회색 처리:

```
[✓] [커] [커] [커] [피] [피]
```

### 5. 키보드 매핑

- 미리보기와 동일한 레이아웃
- 현재 처리해야 할 음절에 해당하는 키는 강조 표시
- 미활성 키는 흐리게 처리

---

## 입력 규칙

키 매핑은 `event.code` 기준으로 처리 (한글 입력 상태에서도 안정적으로 동작):

```
KeyA → 1번째 음절
KeyS → 2번째 음절
KeyD → 3번째 음절
KeyF → 4번째 음절
KeyJ → 5번째 음절
KeyK → 6번째 음절
KeyL → 7번째 음절
Semicolon → 8번째 음절
```

---

## MISS 처리 방식

**틀려도 다음 박자로 자동 진행**한다. (리듬이 끊기지 않도록)

```
MISS 처리 → 해당 음절 실패 기록 → 다음 음절로 이동
```

---

## 게임 종료 조건

| 조건 | 결과 |
| --- | --- |
| 게이지가 0이 됨 | 즉시 GAME OVER |
| 모든 음절 처리 완료 | 다음 스테이지 미리보기로 이동 |
| Stage 15 완료 | ALL CLEAR 결과 화면 |
